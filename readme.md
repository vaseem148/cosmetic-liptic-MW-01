# 💄 Cosmetic Lipstick MW-01

A modern and responsive cosmetic website designed to showcase premium lipstick collections with a luxury user experience.

## 🚀 Features

* 💋 Elegant lipstick showcase
* 🎨 Modern UI with smooth animations
* 📱 Fully responsive design
* ⚡ Built with reusable components
* 🛍️ Product-based layout

## 🛠️ Tech Stack

* React JS
* Tailwind CSS
* Framer Motion

## 📂 Project Structure

* Components (Hero, Categories, Products, Reviews)
* Pages (Home)
* Data (Products)

## 🎯 Use Case

Perfect for:

* Cosmetic brands
* Beauty startups
* Portfolio projects
